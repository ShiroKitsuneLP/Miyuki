# Miyuki 🦊

**Miyuki** is a clever and friendly fox-themed Discord bot built with [discord.js](https://discord.js.org/).  
Bring a touch of fox magic to your server with fun features and easy customization!

> **Note:** Miyuki is still in early development! Many features are planned and coming soon. Stay tuned for updates!

## Requirements

- [Node.js](https://nodejs.org/) v18 or higher
- A Discord bot token ([How to get one?](https://discordjs.guide/preparations/setting-up-a-bot-application.html#creating-your-bot))
- Your server guild ID for command deployment

- Required npm packages:

## Setup

1. **Clone the repository**
   ```sh
   git clone https://github.com/ShiroKitsuneLP/Miyuki.git
   cd Miyuki
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Add your configuration**
   - Create a `config.json` file in the `config` folder:
     ```json
     {
       "discord": {
         "token": "YOUR_DISCORD_BOT_TOKEN"
       }
     }
     ```

4. **Start the bot**
   ```sh
   npm run start
   ```
